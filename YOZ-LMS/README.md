## 🌟 Hakkında

YOZ-LMS, **Google Gemini AI** tarafından desteklenen modern bir öğrenme yönetim sistemidir. Geleneksel LMS platformlarının ötesinde, **otomatik içerik üretimi**, **kişiselleştirilmiş öğrenme yolları** ve **gerçek zamanlı analiz** sunar.

### 🎯 Misyonumuz

Eğitimde teknolojiyi kullanarak öğrenme deneyimini kişiselleştirmek ve herkes için erişilebilir, etkili ve ilgi çekici eğitim materyalleri oluşturmak.

### 💡 Neden YOZ-LMS?

- 🧠 **AI Destekli İçerik:** Gemini API ile otomatik ders materyali üretimi
- 🌍 **Çok Dilli:** 20+ dil desteği ile küresel erişim
- 📊 **Seviye Bazlı:** 4 farklı öğrenme seviyesi (Beginner → Expert)
- ⚡ **Gerçek Zamanlı:** Anlık içerik üretimi ve feedback
- 🎨 **Modern UI:** Tailwind CSS ile responsive, kullanıcı dostu arayüz
- 🔒 **Güvenli:** Enterprise-level güvenlik önlemleri

## 📋 Kurulum

### ⚡ Hızlı Başlangıç

<details>
<summary>👆 Tıkla ve 5 dakikada kurulumu tamamla!</summary>

#### 1️⃣ Prerequisites
- [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) veya [VS Code](https://code.visualstudio.com/)
- [Git](https://git-scm.com/)

#### 2️⃣ Repository Klonlama
```bash
git clone https://github.com/username/YOZ-LMS.git
cd YOZ-LMS
```

#### 3️⃣ API Key Yapılandırması
- [Google AI Studio](https://makersuite.google.com/app/apikey) gidin
- Ücretsiz API key oluşturun
- `YOZ-LMS.Server/appsettings.json` dosyasını düzenleyin:

```json
{
  "Gemini": {
    "ApiKey": "BURAYA_KENDI_API_KEYINIZI_YAZIN"
  }
}
```

#### 4️⃣ Çalıştırma
```bash
# Backend'i başlat
dotnet run --project YOZ-LMS.Server

# Frontend'i başlat (ayrı terminal)
dotnet run --project YOZ-LMS.Client
```

#### 5️⃣ Tarayıcıda Aç
🌐 [http://localhost:5034](http://localhost:5034)

</details>

### 🔌 API Endpoints

| Method | Endpoint | Açıklama |
|--------|-----------|----------|
| `POST` | `/api/learning/generate` | İçerik üretimi |
| `GET` | `/api/learning/languages` | Desteklenen diller |
| `GET` | `/api/learning/levels` | Öğrenme seviyeleri |
| `GET` | `/health` | Health check |

### 📝 API Kullanımı

#### İçerik Üretme
```bash
curl -X POST http://localhost:5223/api/learning/generate \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Python programlama temelleri",
    "language": "Turkish",
    "level": "Beginner"
  }'
```

#### Response Format
```json
{
  "search": "Python programlama temelleri",
  "level": "Beginner",
  "language": "Turkish",
  "results": {
    "content": {
      "header": "Python Programlama Temelleri",
      "detail": "Python, yüksek seviyeli...",
      "sources": ["https://docs.python.org/3/"],
      "questions": {
        "q1": {
          "questionText": "Python kim tarafından geliştirildi?",
          "answer": "Guido van Rossum",
          "explanation": "Python, 1991 yılında..."
        }
      }
    }
  }
}
```
